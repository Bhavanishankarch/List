• Control Statements
    • The statements that are user to control flow of execution.

  • Decision-Making Statements
    • The statements that are select a block of code to be execute or skip depends on condition.
    Example: if,if-else and switch.

  • Iterative or Looping Statments
    • The statements that executes a block of code zero or more times.Example: while, do- while
    and for.
